Variable

define

define既可以定义变量，也可以定义函数。

默认情况下，变量的值是不可变的，它只能绑定和重新绑定。

define 是可以做全局绑定的，每次只能绑定一个变量。

let let*

如果想一次绑定多个变量，可以使用 let，let 是局部绑定。在绑定的过程中，需要互相引用，可以使用 let*
