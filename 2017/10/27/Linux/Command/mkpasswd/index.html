mkpasswd 密码生成

发表于 2017-10-27 | 分类于 Linux ， Command

命令参数

-l 标志定义口令长度。缺省值为9

  • 生成一个20位的密码 mkpasswd -l 20

-c 标志定义口令必须包含的最少小写字母个数。缺省值为2。

-C 标志定义口令必须包含的最少大写字母个数。缺省值也是2。

-p 标志标明设立口令的程序。如果/etc/yppasswd存在的话则它为缺省值，否则就用/bin/passwd。
