Scope

  • compile：默认选项，编译测试运行都有效。项目在编译，测试，运行阶段都需要这个artifact对应的Jar。
  • provided：在编译和测试时有效。在编译测试阶段，我们需要这个artifact对应的jar包在classpath中，而在运行阶段，假定已经提供了这个jar包，所以无需这个artifact对应的Jar。
  • runtime：在测试和运行时有效。
  • test：仅在测试时有效。
  • system：在编译和测试时有效，与本机系统关联，可移植性差。
